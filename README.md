# Book Tracker

**Автор:** Солодянников Тихон
## Описание
Это приложение для учета прочитанных книг. Вы можете добавлять книги, фильтровать их по жанру и количеству страниц, а также сохранять и загружать данные в JSON.

## Инструкция по запуску
1. Установите необходимые библиотеки:
pip install PyQt5


2. Запустите программу:
python main.py


3. Добавляйте книги, используйте фильтры для поиска по жанру и количеству страниц.

## Примеры использования
- Добавление новой книги с названим, автором, жанром и количеством страниц.
- Фильтрация по жанру (например, "Фантастика").
- Отфильтровать книги с более чем 200 страниц.
